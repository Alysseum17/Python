## Phase 1: Python Core (Days 1–5)

**Day 1 – Syntax, Variables, Data Types, Strings**
Conditionals, loops, f-strings, string methods, input/output.

**Day 2 – Functions, Lambdas, Scope, Closures**
Args/kwargs, first-class functions, higher-order functions, lambda expressions, LEGB scope rule.

**Day 3 – Data Structures Deep Dive**
Lists, tuples, sets, dicts. List/dict/set comprehensions, generator expressions, named tuples.

**Day 4 – OOP in Python**
Classes, inheritance, multiple inheritance, MRO, encapsulation, class/static methods, dunder methods, dataclasses, abstract base classes.

**Day 5 – Exceptions, Modules, Type Hints**
Custom exceptions, try/except/finally, import system, creating packages, typing module, Pydantic basics, mypy.

---

## Phase 2: Intermediate Python (Days 6–10)

**Day 6 – Iterators, Generators & Generator Pipelines**
Iterator protocol, yield, generator pipelines for large data streams, `itertools`.

**Day 7 – Decorators & Context Managers**
Writing decorators, decorator factories, `functools.wraps`, `contextlib`, custom context managers.

**Day 8 – File Handling & Serialization**
Text, CSV, JSON, YAML, binary. `pathlib`. Pickle vs JSON. Large file processing patterns.

**Day 9 – Regular Expressions & Text Processing**
`re` module deep dive, common patterns, parsing and cleaning data.

**Day 10 – Virtual Environments, Package Management & Project Structure**
venv, pip, Poetry, uv, pyproject.toml, project layout, ruff, black.

---

## Phase 3: Concurrency & Performance (Days 11–14)

**Day 11 – Threading & Multiprocessing**
GIL, `threading`, `multiprocessing`, `concurrent.futures`, choosing the right model.

**Day 12 – Async Python**
`asyncio`, async/await, `aiohttp`, async generators, event loop.

**Day 13 – Performance & Profiling**
`cProfile`, `timeit`, memory profiling, `collections` (deque, Counter, defaultdict, OrderedDict), optimization patterns.

**Day 14 – Testing & Code Quality**
pytest (fixtures, parametrize, mocking), unittest, doctest, tox, type checking with mypy/pyright.

---

## Phase 4: Databases & SQL in Python (Days 15–18)

**Day 15 – SQL Refresher + SQLite with Python**
`sqlite3`, raw SQL from Python, connection/cursor management.

**Day 16 – SQLAlchemy Core**
Engine, connections, metadata, Table objects, expressions, transactions.

**Day 17 – SQLAlchemy ORM**

**Day 18 – PostgreSQL + Advanced Patterns**
psycopg2/asyncpg, connection pooling, bulk inserts, COPY, indexing.
